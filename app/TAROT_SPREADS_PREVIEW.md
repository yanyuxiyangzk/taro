## 动画效果

### 火焰按钮
```
静态：橙红渐变
悬停：黄橙红发光 + 粒子闪烁
点击：开始牌阵流程
```

### 牌阵卡片
```
静态：扑克牌背景 + 紫色遮罩
悬停：边框高亮 + 渐变光晕
点击：进入牌阵详情
```

## 颜色方案

- **引导文案**: 紫色渐变卡片
- **火焰按钮**: 橙→红→橙渐变
- **标题**: 紫色文字
- **卡片背景**: 紫色→黑色渐变
- **扑克牌符号**: 紫色半透明
- **文字**: 紫白色系
