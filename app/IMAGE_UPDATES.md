## 自定义图片

如果您想使用自己的图片，只需替换 `divinationCards` 数组中的 `bg` URL：

```typescript
const divinationCards = [
  { 
    title: '卡牌问答', 
    desc: '100w+ 回答', 
    icon: '🎴', 
    bg: 'YOUR_IMAGE_URL_HERE' 
  },
  // ...
];
```

## 注意事项

- 图片 URL 使用 `?w=400` 参数优化加载速度
- 所有卡片都应用了渐变遮罩（`from-black via-black/50 to-transparent`）
- 确保图片与主题相关，保持神秘、占卜的氛围
- 建议使用深色或中性色调的图片，以配合应用的紫色主题
