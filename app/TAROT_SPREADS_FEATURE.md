# 塔罗牌阵功能说明

## 新增功能

在占卜页面的神秘之书上方，新增了**塔罗牌阵**功能区域。

## 功能特点

### 牌阵展示

目前提供两种经典塔罗牌阵：

#### 1. 时光流牌阵 ⏳
- **卡牌数量**: 3张
- **描述**: 探索过去、现在与未来的时间轨迹，洞察事件发展脉络
- **难度**: 初级
- **时长**: 5-10分钟
- **适合**: 初学者，快速指引
- **背景**: 时钟/时间主题图片

#### 2. 凯尔特十字 ✝️
- **卡牌数量**: 10张
- **描述**: 最经典的塔罗牌阵，全面解析复杂问题的各个层面
- **难度**: 高级
- **时长**: 15-20分钟
- **适合**: 深度分析，复杂问题
- **背景**: 凯尔特十字架图片

### 界面设计 (已优化 ✅)

✨ **标题栏**
- 左侧：闪光图标 + "塔罗牌阵"标题
- 右侧："更多"按钮 + 右箭头

✨ **牌阵卡片 - 全新设计**
- **布局**: 单列垂直布局（从2列改为1列）
- **背景**: 半透明背景图 + 紫色渐变遮罩
- **神秘粒子**: 4个闪烁光点动画
- **图标容器**: 16x16圆形，渐变背景，旋转边框动画

✨ **卡片信息**
- **标题**: 更大字体 (text-lg) + 加粗
- **徽章**: 渐变背景 + 阴影效果
- **描述**: 更详细的牌阵说明
- **元数据**: 难度等级 + 预计时长

✨ **动画效果**
- 粒子闪烁动画（不同延迟）
- 悬停时图标边框旋转（3秒一圈）
- 顶部/底部光线效果
- 整体发光效果
- 箭头颜色渐变

### "更多"功能

点击右上角的"更多"按钮可以：
- 查看所有可用的塔罗牌阵
- 浏览更多牌阵类型
- 了解每种牌阵的详细说明
