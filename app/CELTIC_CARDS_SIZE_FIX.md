# 凯尔特十字卡片尺寸修复 ✝️

## 问题

卡片太小，几乎看不见，叠加在一起更不清楚。

## 解决方案

1. 改为横排布局（不再叠加）
2. 增大卡片尺寸
3. 使用轻微旋转效果

## 改进详情

### 1. 布局变化

#### 原版（叠加布局）
```jsx
<div className="relative h-40">
  {/* 左卡片 */}
  <div className="absolute -translate-y-2 -rotate-12" />
  
  {/* 中心卡片 */}
  <div className="absolute z-10" />
  
  {/* 右卡片 */}
  <div className="absolute translate-y-2 rotate-12" />
</div>
```

**问题**:
- 卡片叠在一起
- 看不清楚
- 容器高度限制

#### 新版（横排布局）
```jsx
<div className="flex items-center justify-center gap-2">
  {/* 左卡片 */}
  <div className="transform -rotate-6" />
  
  {/* 中心卡片 */}
  <div />
  
  {/* 右卡片 */}
  <div className="transform rotate-6" />
</div>
```

**改进**:
- ✅ 横排显示
- ✅ 每张卡片都清晰可见
- ✅ 轻微旋转增加动感

### 2. 卡片尺寸

#### 原版
```jsx
w-24 h-36  // 96x144px
```

#### 新版
```jsx
w-[70px] h-[105px]  // 70x105px
```

**说明**:
- 虽然单个卡片略小
- 但横排布局使整体更大
- 3张卡片总宽度: 70×3 + 间距 = 约220px
- 比叠加布局视觉效果更好

### 3. 旋转角度

#### 原版
```jsx
-rotate-12  // -12度
rotate-12   // +12度
```

#### 新版
```jsx
-rotate-6   // -6度
rotate-6    // +6度
```

**改进**:
- ✅ 更轻微的旋转
- ✅ 不会太倾斜
- ✅ 保持整洁

### 4. 间距

```jsx
gap-2  // 8px 间距
```

**效果**:
- 卡片之间有适当间距
- 不会太挤
- 视觉舒适

## 技术实现

### Flex 布局
```jsx
<div className="flex items-center justify-center gap-2">
```

**属性**:
- `flex`: 弹性布局
- `items-center`: 垂直居中
- `justify-center`: 水平居中
- `gap-2`: 8px 间距

### 固定尺寸
```jsx
className="w-[70px] h-[105px]"
```

**说明**:
- 使用方括号指定精确尺寸
- 不受 Tailwind 默认尺寸限制
- 70x105px 是理想比例

### 轻微旋转
```jsx
className="transform -rotate-6"
className="transform rotate-6"
```

**效果**:
- ±6度旋转
- 增加动感
- 不会太倾斜
